Gruppo UniveMean

Istruzioni per compilare il server:
    - Recarsi nella directory /server
    - Eseguire il comando npm install per installare tutti i pacchetti necessari
    - Eseguire il comando tsc battleship.ts per compilare i sorgenti TypeScript in JavaScript
    - Eseguire il comando node battleship per eseguire il server

Istruzioni per compilare il client:
    - Recarsi nella directory /client/battleship
    - Eseguire il comando npm install per installare tutti i pacchetti necessari
    - Eseguire il comando ng build per compilare il client in angular
    - Opzionale: eseguire il comando ng serve --open per aprire il client in automatico
